# 📢 COMMANDES DISPONIBLES

## Shoutout

```text
!so pseudo
```

Exemples :

```text
!so benjyyi
!so @benjyyi
```

---

## Fonctionnement

```text
Moderator
↓
!so pseudo
↓
Twitch API
↓
Annonce Twitch
↓
Shoutout Officiel Twitch
```

---

## Conditions du Shoutout Officiel

Twitch exige :

```text
✅ Stream en live
✅ Au moins 1 viewer
✅ Scopes corrects
✅ Moderator ID valide
✅ Permissions modérateur
```

---

## Exemple de message

```text
👑 Le royaume recommande fortement une visite chez benjyyi.

⚔️ Les éclaireurs de Godas signalent que benjyyi continue ses activités sur Twitch.

📡 Transmission interceptée : benjyyi a récemment été aperçu dans Sea of Thieves.
```

Les messages sont choisis aléatoirement.
